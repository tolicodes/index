# new-project

## Template

1. Go to https://github.com/new and enter a project name
2. Add "Node.js" `.gitignore` and choose `GNU General Public License`
3. Clone it down
4. Open it in VS Code
5. Copy [new-project](./new-project)
6. Run `yarn`
